# 프리미엄 기능 명세

이 문서는 향후 구현 예정인 프리미엄 전용 기능들을 정리합니다.

## 1. 실거래가로 내 집 시세 찾기

### 개요
국토교통부 실거래가 API를 연동하여 사용자의 주택 시세를 자동으로 조회하는 기능입니다.

### 상태
- **분류**: 프리미엄 전용 기능
- **우선순위**: 높음
- **구현 상태**: 계획 단계

### 기능 설명
사용자가 주소를 입력하면 국토교통부 실거래가 데이터베이스에서 해당 지역의 최근 거래 내역을 조회하여:
1. 유사 평형대의 최근 실거래가 표시
2. 평균 시세 자동 계산
3. 시세 추이 그래프 제공
4. 해당 값을 시뮬레이터에 자동 입력

### 기술 요구사항
- **API**: 국토교통부 실거래가 공개 API
- **인증**: API 키 필요 (환경변수로 관리)
- **데이터 처리**: 
  - 주소 → 법정동코드 변환
  - 최근 3개월 거래 데이터 조회
  - 평형대별 평균가 계산
- **UI 컴포넌트**:
  - 주소 검색 자동완성
  - 실거래가 목록 표시
  - 시세 추이 차트

### 구현 계획
1. **Phase 1**: API 연동 및 기본 조회 기능
2. **Phase 2**: 데이터 시각화 (차트, 그래프)
3. **Phase 3**: 프리미엄 결제 시스템 연동
4. **Phase 4**: 사용자 히스토리 저장 기능

### 참고 사항
- 무료 사용자에게는 기능 존재를 알리되 "프리미엄 전용" 배지 표시
- 프리미엄 사용자만 실제 API 호출 가능
- API 호출 제한을 고려한 캐싱 전략 필요
